What is MP3 & How Does it Work?

Officially known as MPEG-1 Audio Layer III or MPEG-2 Audio Layer III, MP3 is a standard digital coding format for audio.

The format compresses files by discarding data that the algorithm considers imperceptible, making them smaller. This is called lossy compression because it involves the loss of data.
